# Web3 dApp（MetaMask + ethers + Next.js）

## 中文版
- 项目功能：支持钱包连接、资产查询、ETH 转账、消息签名与网络切换，并通过服务端代理解决 DeepSeek API 的跨域问题。
- 技术框架：Next.js 14、React 18、TypeScript、ethers.js v6、Zustand（localStorage 持久化）、TailwindCSS、pnpm。
- 亮点/难点：SSR/CSR 兼容的客户端钱包交互；服务端代理解决第三方 API 的 CORS 并保护密钥。

## English Version
- What it does: Connect wallet, show assets, transfer ETH, sign messages, switch networks, and proxy DeepSeek API to avoid CORS.
- Tech used: Next.js 14, React 18, TypeScript, ethers.js v6, Zustand (persist to localStorage), TailwindCSS, pnpm.
- Highlights/Challenges: SSR/CSR‑safe wallet interactions; server proxy resolves third‑party API CORS and secures the API key.
